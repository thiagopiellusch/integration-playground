# Integration Playground

Repositório dedicado à exploração de padrões de integração, APIs e arquitetura corporativa.

Este espaço representa minha visão sobre integração de sistemas em ambientes complexos.

## Temas abordados
- APIs REST e GraphQL
- Webhooks e eventos
- Integrações corporativas (ERP, CRM, legados)
- Arquitetura orientada a eventos
- Governança de integrações
